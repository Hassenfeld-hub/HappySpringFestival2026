# 2026 丙午马年 · 新春快乐 🧧

一个2026农历新年（马年）互动贺卡及在线许愿墙。该项目结合了动态烟花特效以及基于 Supabase 的实时愿望寄送功能。

## 🌟 项目预览

*   **书法展示**：动态布局的“新春快乐”书法字样，配合传统印章设计。
*   **烟花特效**：基于 HTML5 Canvas 的粒子物理模拟，支持点击交互与自动绽放。
*   **许愿墙**：实时读取与提交愿望，采用红色信笺风格展示。
*   **响应式设计**：完美适配移动端与桌面端。

## 🚀 核心功能

1.  **书法美学**：使用 Google Fonts `Ma Shan Zheng` 书法字体，结合 SVG 滤镜模拟真实印章质感。
2.  **交互式烟花**：点击屏幕任何位置可触发彩色烟花粒子效果，支持重力感应、摩擦力及尾迹模拟。
3.  **数据持久化**：使用 **Supabase** 作为后端数据库，实现愿望的实时存储与展示。
4.  **访问统计**：集成不蒜子（Busuanzi）统计，记录页面浏览量（烟花点亮人数）。

## 🛠️ 技术栈

*   **前端**：HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
*   **动画**：Canvas API
*   **字体**：Google Fonts (Ma Shan Zheng)
*   **后端/数据库**：Supabase (PostgreSQL)
*   **统计**：Busuanzi.js
